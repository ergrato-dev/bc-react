# 📘 Semana 08: Gestión de Estado Global con Zustand

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Comprender el problema del estado global en aplicaciones React
- ✅ Implementar stores con Zustand y TypeScript
- ✅ Crear acciones y selectores tipados
- ✅ Persistir estado en localStorage
- ✅ Implementar middleware personalizado
- ✅ Aplicar patrones de organización de stores
- ✅ Integrar Zustand con React Router

---

## 📚 Requisitos Previos

- Semana 07 completada (React Router v6)
- Conocimiento de hooks de React (useState, useEffect)
- TypeScript intermedio (genéricos, interfaces)
