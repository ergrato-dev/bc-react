## 🎓 Resultados de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Configurar React Router v6 en proyectos TypeScript

- ✅ Crear rutas dinámicas con parámetros tipados
- ✅ Construir layouts reutilizables con Outlet
- ✅ Manejar query strings con useSearchParams
- ✅ Implementar rutas protegidas con autenticación
- ✅ Usar useNavigate para navegación programática
- ✅ Gestionar estados de carga con loader/action
