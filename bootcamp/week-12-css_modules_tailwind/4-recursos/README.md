## 🛠️ Bibliotecas Complementarias

### Para CSS Modules

```bash
# Utilidad para combinar clases condicionalmente
pnpm add clsx
# Alternativa popular
pnpm add classnames
```

### Para Styled Components

```bash
# Styled components y tipos
pnpm add styled-components
pnpm add -D @types/styled-components
# Polished: utilidades de CSS
pnpm add polished
```

### Para Tailwind CSS

```bash
# Tailwind y plugins
pnpm add -D tailwindcss postcss autoprefixer
# CVA para variantes tipadas
pnpm add class-variance-authority
# Merge de clases Tailwind
pnpm add tailwind-merge
# clsx + tailwind-merge
pnpm add clsx tailwind-merge
```

## 📋 Comparativas y Benchmarks

### Comparativa de Tamaño de Bundle

| Estrategia            | Runtime JS       | Bundle CSS        | Rendimiento |
| --------------------- | ---------------- | ----------------- | ----------- |
| **CSS Modules**       | ❌ Sin runtime   | ✅ Solo CSS usado | ⭐⭐⭐⭐⭐  |
| **Styled Components** | ⚠️ ~12KB runtime | ✅ CSS-in-JS      | ⭐⭐⭐⭐    |
| **Tailwind CSS**      | ❌ Sin runtime   | ✅ Purge no usado | ⭐⭐⭐⭐⭐  |

### Cuándo Usar Cada Estrategia

| Criterio                         | CSS Modules  | Styled Components | Tailwind    |
| -------------------------------- | ------------ | ----------------- | ----------- |
| **Equipo familiarizado con CSS** | ✅ Ideal     | ⚠️ Curva          | ⚠️ Curva    |
| **Design system complejo**       | ⚠️ Posible   | ✅ Ideal          | ✅ Bueno    |
| **Prototipado rápido**           | ⚠️ Más lento | ⚠️ Medio          | ✅ Ideal    |
| **Máximo rendimiento**           | ✅ Ideal     | ⚠️ Runtime        | ✅ Ideal    |
| **Theming dinámico**             | ⚠️ CSS vars  | ✅ Ideal          | ⚠️ CSS vars |

## 📌 Checklist de Aprendizaje

### CSS Modules

- [ ] Entender el scope local automático
- [ ] Usar `composes` para composición
- [ ] Integrar `clsx` para clases condicionales
- [ ] Configurar TypeScript declarations

### Styled Components

- [ ] Crear componentes styled básicos
- [ ] Usar props para estilos dinámicos
- [ ] Implementar theming con ThemeProvider
- [ ] Manejar props transient (`$prop`)

### Tailwind CSS

- [ ] Entender utility-first
- [ ] Configurar tema personalizado
- [ ] Usar CVA para variantes
- [ ] Combinar con `tailwind-merge`
