## 🎨 Comparativa de Estrategias

| Característica | CSS Modules          | Styled Components | Tailwind CSS          |
| -------------- | -------------------- | ----------------- | --------------------- |
| Scope          | Local automático     | Local automático  | Global con utilidades |
| TypeScript     | Requiere tipos       | Nativo            | Requiere config       |
| Bundle size    | Solo CSS usado       | Runtime JS        | Purge automático      |
| Dynamic styles | Clases condicionales | Props nativas     | Clases condicionales  |
| Learning curve | Baja                 | Media             | Media                 |
| Ideal para     | Apps medianas        | Design systems    | Prototipado rápido    |
