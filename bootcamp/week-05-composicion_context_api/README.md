# 📘 Semana 05: Composición de Componentes y Context API

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Aplicar patrones de composición para crear componentes reutilizables
- ✅ Usar children y render props para flexibilidad en componentes
- ✅ Implementar el patrón Compound Components
- ✅ Entender y aplicar Context API con TypeScript
- ✅ Crear providers tipados para compartir estado global
- ✅ Optimizar Context para evitar re-renders innecesarios
- ✅ Combinar múltiples contextos efectivamente
